Carbon Emission Analysis Project (SQL)
📌 Project Overview

Understanding carbon emissions across industries is essential to combat climate change. This project analyzes carbon emission data using SQL to identify high-polluting sectors, track emission trends, and support sustainable decision-making.

Source: Kaggle (Carbon Emissions by Sector Dataset)
The dataset includes:
1. Industry/Sector
2. Country
3. Year
4. Emission values
5. Greenhouse gas types (CO2, CH4, N2O)

Tools & Technologies
1. SQL (MySQL Workbench)
2. Database Management System (RDBMS)
3. Data Analysis using SQL Queries

Key Insights

1 Tools & Technologies
SQL (MySQL Workbench)
Database Management System (RDBMS)
Data Analysis using SQL Queries
2. Detected industries with declining emission patterns.
3. Compared emission distribution across continents.
4. Highlighted top 3 polluting industries and their yearly trends.

Conclusion

This project demonstrates how SQL can be used effectively for environmental data analysis. It provides insights that can help policymakers and organizations make informed decisions for reducing carbon emissions.
